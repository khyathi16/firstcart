# 🛒 FirstCart

FirstCart is a Full Stack E-Commerce Web Application inspired by Flipkart.

## 🚀 Features

- User Login & Registration
- Product Search
- Product Details Page
- Add To Cart
- Wishlist
- Remove From Cart
- Payment Page
- Cash On Delivery Option
- Order Success Page
- Responsive Design
- MongoDB Database Integration

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
- Mongoose
